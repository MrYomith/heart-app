# Mio character assets

Drop the 3D Mio character renders here as **transparent PNG** (≈512×512, 2×/3× optional).
The app loads them automatically — no code change needed. Until a file exists, that
pose falls back to a soft emoji badge.

These are the exact poses used in the mockups. File names must match exactly:

| File | Pose (from the mockups) | Used on |
|---|---|---|
| `mio_default.png`   | Standing, hand on chest | Home hero, generic |
| `mio_happy.png`     | Winking, thumbs-up | Celebrations, "doing great" banners |
| `mio_calm.png`      | Hands on heart / calming | Emotional check-in, support |
| `mio_scrubs.png`    | Surgery cap + scrubs, hands together | Surgery Day screen |
| `mio_celebrate.png` | Arms up with confetti | Milestones, completion |
| `mio_thriving.png`  | Lotus meditation, serene (hoodie) | Thriving stage |
| `mio_reading.png`   | Holding an open book | Education Hub |
| `mio_clipboard.png` | Holding the daily-plan clipboard | Today's Plan |
| `mio_waving.png`    | Friendly wave | Home footer, greetings |
| `mio_sleeping.png`  | Resting / sleeping | Inpatient footer, sleep |
| `mio_walking.png`   | Walking in trainers | Rehab / activity |

Source: the same renders used in the MioHeart design mockups. Provide as PNG (and SVG
if available). Once added here, run `flutter pub get` and the character appears across
every screen exactly as designed.
